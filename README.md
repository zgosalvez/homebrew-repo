# Homebrew Repo

A collection of software packages that do not pass Homebrew's requirements.

## Formulae

* _None_

### How do I install these formulae?
`brew install zgosalvez/repo/<formula>`

Or `brew tap zgosalvez/repo` and then `brew install <formula>`.
